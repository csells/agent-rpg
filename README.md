# agent-rpg

An RPG-inspired skill tree for leveling up engineers in agentic coding.

## Concept

A progression system modeled after RPG skill trees, designed to help
engineers systematically develop and demonstrate proficiency with AI
coding agents. Think character classes, skill points, and boss fights —
but for shipping software with Claude, Codex, Gemini, and whatever
comes next.

## Status

Early stage. Vision doc and requirements coming soon.

## What's Coming

- **Skill tree definition** — branching paths for different agentic
  coding disciplines (prompt engineering, agent orchestration, tool
  design, review workflows, etc.)
- **Level-up criteria** — concrete, measurable milestones for each node
  in the tree
- **Vision doc** — full design rationale and pedagogy
- **Requirements** — implementation spec
